# PolyCraft 3D : 3D 拟真空间看房交互系统
> 基于 Jetpack Compose 与 Kotlin 纯本地算法构建的第一人称 3D 拟真室内空间预览与户型定制系统。

---

## 🎨 视觉风格与美术主题
本项目完美契合 **“极简纯粹主义 (Clean Minimalism)”** 美学理念：
- **画幅留白**：采用大面积松散、透气的负空间布局，过滤视觉噪音，使用户能全身心沉浸于 3D 空间的穿行之中。
- **色彩规范**：严格承袭双态 Material Design 3 色调。背景选用高贵温润的深邃星空灰（`#211F26`），线条与边缘辅以极简紫（`#D0BCFF`）、薄荷嫩绿（`#B5E48C`）和阳光暖黄（`#FFD54F`）以指示不同的交互焦点与智能电器节点。
- **排版考究**：指示文本选用 Space Grotesk 等粗重比率无衬线标题，坐标、偏航角等状态信息则采用等宽字体（`JetBrains Mono`）精细化浮动排布，带来兼具建筑工程蓝图与未来感的设计张力。

## 🚀 核心功能亮点

### 1. 第一人称相机与物理平移机制 (3 Player Controller Engine)
- **自由视线观察**：用户在 3D 画布区域滑动手指，即可控制虚拟摄像机的 **偏航角 (Yaw/水平视角)** 与 **俯仰角 (Pitch/垂直视域)**，实现广角视野观察。
- **视线对齐平移 (D-Pad Controller)**：设计了高灵敏度的微型游戏罗盘键，前、后、左、右所有平移算法均与摄像机视口方向对齐。前行、后退自动沿前进向量方向深入，确保极佳的室内代入感。
- **边界碰撞限制**：每个独立房间边界都设有物理碰撞墙障（4.5f~5.2f 边界包围盒），防止玩家意外穿过外墙掉入三维黑洞，大幅度提高空间探索真实感。

### 2. 空间平稳飞行过渡 (Linear Ease-Out Space Flier)
- 提供 4 大现代样板间一键滑跃切换功能。在点击诸如 **“阳台”**、**“卧室”** 等选项卡时，系统绝不采用生硬的即时传送，而是启动基于协程（Coroutines）的多帧高频插值引擎。
- 支持偏航、俯仰与三维位置的并行多波段渐变（Easing Line），模拟直升机漫游般的顺滑运镜效果。

### 3. 三维交互热点与智能节点 (Interactive Hotspots Interface)
- 每个空间都具有独特的橙色星芒热点（如客厅的巨幕电视、卧室的智能台灯等）。
- 用户点击热点，将浮现高反差黑色磨砂玻璃特效面板，展现详细的软装设计说明。
- 支持 **实时设备操控**：
  - 点击 **“电视热点”**：一键开/关 3D 投影电视墙，开启时在空间中渲染霓虹流光扫描线动态画幅，关闭时回归镜面质感。
  - 点击 **“台灯热点”**：远程控制床头暖气灯光的启闭，开启时提供多层次黄金光晕漫反射。

### 4. 个性化照片壁画实时敷贴 (Personal Image Customizer Frame)
- 在 **“书房”** 的侧墙上挂设了壁挂相屏。用户可通过顶部的 **“拍照建模”**、**“相册导入”** 加载个人照片，算法会将照片直接绘制并贴紧投射于书房的三维照片框内！
- 导入历史会被安全存档于本地 **Room 数据库** 中，支持历史卡片切换并支持一键擦除旧设计，实现极具人情味的个性化居家预览。

### 5. 整屋空间精装定制面板 (Architectural Customizer Deck)
- **墙体涂刷调色**：提供 Ivory（经典象牙白）、Sage（薄荷微晶）、Charcoal（深邃炭灰）与 Lavender（温润紫泥）4 种主流艺术漆一键涂布。
- **地面纹理拼设**：支持柚木人字拼木地板、大师级希腊雅士白大理石、以及数字几何霓虹格栅。
- **户外日照亮度调节**：高保真日光漫射系数无级微调。
- **工业级设计输出分享**：支持将整屋设计的偏航俯航、材质选用、电器通断等数据一键导出并分享为 `.TXT` 蓝图或 `.XML` 参数代码。

## 📐 三维渲染与透视投影数学公式说明
对于空间内任意一点 $P(x, y, z)$，当虚拟机位处于 $Cam(x_c, y_c, z_c)$ 且横纵观察角为 $\theta$ (Yaw), $\phi$ (Pitch) 时：

1. **绝对坐标相机平移 (Camera Translation)**:
   $$x' = x - x_c$$
   $$y' = y - y_c$$
   $$z' = z - z_c$$

2. **视线角坐标偏航变换 (Yaw Rotation around Y-axis)**:
   $$x'' = x' \cos(-\theta) - z' \sin(-\theta)$$
   $$z'' = x' \sin(-\theta) + z' \cos(-\theta)$$
   $$y'' = y'$$

3. **视线角坐标俯仰变换 (Pitch Rotation around X-axis)**:
   $$x''' = x''$$
   $$y''' = y'' \cos(-\phi) - z'' \sin(-\phi)$$
   $$z''' = y'' \sin(-\phi) + z'' \cos(-\phi)$$

4. **近裁剪面过滤与视差透视变换 (Perspective Projection Mapping)**:
   若 $z''' < 0.1$，该点被剔除（属于相机后方）。否则其屏幕投射物理坐标为：
   $$ScreenX = \frac{W}{2} + x''' \cdot \frac{Scale \cdot Zoom}{z'''}$$
   $$ScreenY = \frac{H}{2} + y''' \cdot \frac{Scale \cdot Zoom}{z'''}$$
   *(其中 $Scale$ 基于屏幕短边自适应缩放，$Zoom$ 为手势焦距放大因子)*。使用画笔绘制多组相互连接的前述 screen 点，即可得到完全透视正确的 3D 拟真空间画面。
